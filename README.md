##  API
- ### onPullDown: () => Promise<any>      
下拉事件
	 
- ### onPullUp: () => Promise<any>  
上拉事件  
 
- ### disablePullUp: boolean   
禁用上拉事件  
 
- ### disablePullDown: boolean   
禁用下拉事件   
 
- ### pullDownText: string    
下拉状态提示文案   

- ### pullUpText: string    
上拉状态提示文案
